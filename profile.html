Why these muscles?
Chest adds width, arms add definition, and legs trigger the most whole-body anabolic growth. As an ectomorph these three groups give you the fastest visible results.
Volume progression
Start at the lower end of each rep range in week 1. Add one set to each major compound lift every 2 weeks. Increase weight when you can comfortably hit the top rep count for all sets.
Your first 4 weeks
Week 1 — Learn the moves
You're brand new, so weight doesn't matter yet — form does. Use light weight (something you could do 15+ reps with) and stop each set 2-3 reps before it gets hard. Use the step-by-step photos and cues on every exercise. Missing a day is fine; missing good form isn't.
Week 2 — Build the habit
Same weights as week 1, but now push each set closer to failure (1-2 reps left in the tank). Focus on showing up for all 4 sessions and getting used to the rest timer and full routine — consistency matters more than intensity right now.
Week 3 — Start adding weight
For any lift where you hit all sets at the top of the rep range last week, add a small jump (2.5-5kg, or the next plate/pin) this week. Log every set in the app so you can see it happening.
Week 4 — Full progressive overload
Follow the plan's listed sets/reps/rest as written from here on. The rule going forward: whenever you complete every set at the top of a rep range, add weight next session. This is also a good week to set your first milestones from the Progress tab.
